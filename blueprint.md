# Project Blueprint

## Overview

MBTI 성격유형 테스트 웹사이트. 20문항을 통해 사용자의 MBTI 유형을 진단하고 상세한 설명을 제공합니다. 티스토리 블로그에 iframe으로 임베드 가능하도록 설계되었습니다.

## Features

- **홈 화면:** MBTI 테스트 소개 및 시작 버튼
- **퀴즈 화면:** 20문항 (7점 Likert 척도), 진행률 표시, 무작위 문항 순서
- **결과 화면:** MBTI 4자리 유형, 유형 설명, 4가지 지표별 세부 점수 막대 그래프
- **공유 기능:** Web Share API / 클립보드 복사 지원
- **반응형 디자인:** 모바일 최적화 (티스토리 임베드 고려)

## Tech Stack

- HTML/CSS/JavaScript (순수, 프레임워크 없음)
- Google Fonts (Noto Sans KR)
- Firebase Hosting (배포)
